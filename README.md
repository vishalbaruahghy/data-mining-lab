# 📊 Data Mining Lab

This repository contains all assignments and practical implementations for the Data Mining Lab course. It includes implementations of supervised and unsupervised learning algorithms using Python and Jupyter Notebooks.

## 📁 Repository Structure

```
data-mining-lab/
├── downloads/
│   ├── readme
│   ├── SMSSpamCollection
│   └── smsspamcollection.zip
├── Supervised-Learning/
│   ├── mnist_number_recog.ipynb
│   └── svm_cancer_classification.ipynb
├── Unsupervised-Learning/
│   ├── knn-iris.ipynb
│   └── Naive_Bayes-spam-classifier.ipynb
├── .gitignore
├── README.md
└── requirements.txt
```

## 🎯 Objectives

- Understand core concepts of Data Mining
- Implement machine learning algorithms using real datasets
- Gain hands-on experience with classification, clustering, data preprocessing, and model evaluation

## 📘 Experiments & Assignments

### 🔹 Supervised Learning

**MNIST Number Recognition**
- Digit classification using machine learning models
- Dataset: MNIST

**SVM Cancer Classification**
- Classify tumors as benign or malignant
- Includes confusion matrix and performance metrics

### 🔹 Unsupervised Learning

**KNN on Iris Dataset**
- Classification using K-Nearest Neighbors
- Dataset: Iris

**Naive Bayes Spam Classifier**
- Email/SMS spam detection
- Dataset: SMS Spam Collection

## ⚙️ Technologies Used

- Python 🐍
- Jupyter Notebook 📓
- Scikit-learn
- NumPy, Pandas
- Matplotlib

## 📊 Key Concepts Covered

- Data preprocessing and feature selection
- Model training & testing
- Confusion matrix, accuracy, precision, recall, F1-score
- Supervised vs unsupervised learning
